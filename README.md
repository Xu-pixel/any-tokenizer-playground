## Features

- ⚡️ Built with Vue 3 + Vite
- 🎨 Instant UI with UnoCSS
- 🦾 TypeScript support
- ✅ Unit testing with Vitest
- ☁️ Deploy on Netlify

## Development

```bash
# install dependencies
pnpm i

# start dev server
pnpm dev

# build for production
pnpm build

# preview production build
pnpm preview
```
